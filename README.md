# Back-churn-analysis1
This project provides a comprehensive, multi-method analysis of customer behaviour and attrition using the Bank Churners dataset. Through the integration of unsupervised clustering, regression modeling, and classification, it presents both interpretive and predictive insights into patterns of customer engagement and churn.

## Objectives
- Identify variables associated with churn
- Build predictive models
- Evaluate model performance

## Tools Used
- R
- dplyr, ggplot2, caret, randomForest

## Files
- Customer Churn Analysis code.Rmd → full analysis
- Customer Churn Analysis Report.pdf → summarized report

## How to Run
1. Download dataset
2. Open Customer Churn Analysis code.Rmd in RStudio
3. Run all chunks
